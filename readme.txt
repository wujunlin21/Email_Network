write a MapReduce program in Java to calculate the maximum of the weights of all outgoing edges for each node in the graph.

Data file stores a list of edges as tab-separated-values.
sample input:
src tgt weight
51 117 1
51 194 1
51 299 3
151 230 51
151 194 79
130 51 10

sample output:
51 3
151 79
130 10
